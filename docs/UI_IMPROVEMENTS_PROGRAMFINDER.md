## Key Changes

### 1. Larger H1 Title
- Change from `text-2xl` to `text-3xl` or `text-4xl`
- More prominent and readable

### 2. Answers Summary Section (NEW)
- Add a collapsible "Your Answers" section above the question card
- Shows all answered questions in a compact format
- Click to expand/collapse for details
- Green checkmarks for answered questions
- Grey dashes for skipped/unanswered

### 3. Prominent Generate Button
- Move button to top-right of answers summary section
- Larger, more visible
- Sticky position when scrolling (optional)

### 4. Compact Layout
- Reduce padding/margins slightly
- Make navigation dots smaller
- Ensure everything fits in viewport

### 5. Visual Improvements
- Better spacing
- Clearer visual hierarchy
- Answers summary uses card design
